Parancsok Jelentései
Angular telepítése
npm install -g @angular/cli Telepíti az Angular-t
Angular fontosabb parancsok
ng new projectname Létrehozza az Angular projektet
ng serve Elindítja az Angular projektet
ng generate component componentname Létrehozza a komponenst
Angular html parancsok
ngIf Ha a feltétel igaz, akkor megjelenik a tartalom
ngFor Az ngFor segítségével tömb elemeit tudjuk megjeleníteni
ngSwitch Az ngSwitch segítségével több feltételt tudunk vizsgálni
